## 기본 점화식과 폐쇄형
1. T(n) = T(n-1) + Θ(1), T(1)=Θ(1) → Θ(n)
2. T(n) = T(n-1) + Θ(n), T(1)=Θ(1) → Θ(n^2)
3. T(n) = T(n/2) + Θ(1), T(1)=Θ(1) → Θ(logn)
4. T(n) = T(n/2) + Θ(n), T(1)=Θ(1) → Θ(n)
5. T(n) = 2T(n/2) + Θ(1), T(1)=Θ(1) → Θ(n)
6. T(n) = 2T(n/2) + Θ(n), T(1)=Θ(1) → Θ(nlogn)
